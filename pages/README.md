# SauceDemo Playwright Automation Framework

## Project Overview

This project automates SauceDemo using Playwright and JavaScript.

## Features

* Page Object Model (POM)
* Data Driven Testing
* Fixtures
* Login Validation
* Cart Validation
* Checkout Flow
* Assertions

## Project Structure

pages/

* LoginPage.js
* InventoryPage.js
* CheckoutPage.js

fixtures/

* Fixtures.js

testdata/

* users.js

tests/

* login.spec.js
* cart.spec.js
* checkout.spec.js

## Installation

npm install

## Run Tests

npx playwright test

## Technologies Used

* Playwright
* JavaScript
* Node.js
